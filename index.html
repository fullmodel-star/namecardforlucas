Gemini API Key 申請教學（完全免費）
1
前往 Google AI Studio
點下方連結，用你的 Google 帳號登入。
開啟 Google AI Studio
2
點「建立 API 金鑰」
進入頁面後，點藍色按鈕 「建立 API 金鑰 (Create API Key)」，選擇現有的 Google Cloud 專案或建立新專案，系統會自動產生一組金鑰。
3
複製 API Key
金鑰格式為 AIzaSy... 開頭，點旁邊的複製按鈕複製起來。
4
貼到上方輸入框，按「儲存」
回到本 App，把金鑰貼入上方輸入框，按 「儲存」，再按 「測試連線」 確認正常運作。完成！
